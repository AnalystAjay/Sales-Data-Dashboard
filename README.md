# 📊 Retail Sales Dashboard

## 📌 Overview
This project is an interactive retail analytics dashboard built using Streamlit and SQL.

## 🚀 Features
- KPI Metrics (Sales, Profit, Quantity)
- Interactive Filters (Month, Country)
- Charts (Bar, Pie, Line, Map)
- Real-time Data Handling

## 🧠 Tech Stack
- Frontend: Streamlit
- Backend: Python, SQL
- Visualization: Plotly

## ▶️ Run Locally
```bash
streamlit run app.py
